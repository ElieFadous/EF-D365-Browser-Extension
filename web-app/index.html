How It Works

Bookmarklet launcher

Clicking the bookmark appends launcher.js (hosted here on GitHub Pages) to the D365 document. It runs in the D365 origin, so all API calls are same-origin with credentials included.

Tool iframes

Each tool opens as an iframe served from github.io. To reach D365 APIs they send a postMessage to the launcher, which relays the fetch and posts the result back.

No extension required

No Chrome extension, no admin approval, no service worker. The only browser permission needed is the ability to save a bookmark — which every browser already allows.

Config in localStorage

All environment config is stored in your browser's localStorage under ef_ppt_config. Nothing is sent to any server. Export to keep a backup.
